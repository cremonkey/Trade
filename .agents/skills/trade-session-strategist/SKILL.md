# SKILL: GoldEdge Session Strategist (v4.4)

## Description
Timing layer for XAU/USD. Use this skill to classify the current market window according to the project timezone and the v4.4 operating framework.

## Instructions
1. Interpret session windows in the project operating timezone unless the root documents explicitly say otherwise.
2. Approved v4.4 windows:
   - Sydney observation: 00:00 - 02:00
   - London AM kill zone: 07:00 - 11:00
   - NY AM kill zone: 14:30 - 17:00
3. Never authorize a trade on price alone. It must be price plus time plus structure.
4. If an older rule file disagrees with the root v4.4 roadmap, the roadmap wins.
